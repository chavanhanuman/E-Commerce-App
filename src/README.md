
# E-Commerce React Typescript App

This Project is a responsive e-commerce web application built using React,React-Roouter-DOM, useRef, Typescript, React Router, Context API, Hooks,HTML, CSS, and Cypress Testing.

# Product and category data are fetched dynamically from the  Fake Platzi Store API.
Filter products by multiple categories.


# Project Information

  version : "0.1.0",

## Running Project From Github:   ##

# Clone The Repository

git clone <https://github.com/chavanhanuman/E-Commerce-App.git>

# Navigate The Project Directory

cd ecommerce-app

# Install Dependencees
 
 npm install

# Start The Server
 
 npm start

# Open Browser
The Application Run
http://localhost:3000

# Run Cypress Testing


## Open Cypress:   ## 

Enter Command: npx cypress open

Enter Command: npx cypress run

## Features ##

1. View all products on the home page.
2. Open a product to see its details.
3. Filter products by category.
4. Sort products as needed.
5. Add products to the cart.
6. Remove products from the cart.
7. See the total number of items and total price in the cart.
8. Keep cart data saved after page refresh.
9. Keep selected filters and sorting after refresh or browser navigation.
10. Tested application functionality using Cypress.
